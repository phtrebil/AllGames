# AllGames
aplicação que consome uma api para exibição de catálogo de jogos.

<h2>Descrição do Projeto</h2>
<p>>Na aplicação principal, criamos uma funcionalidade de busca de jogos utilizando uma API externa que retorna informações de jogos no formato JSON.
  Nossa tarefa principal foi a desserialização desses objetos JSON em objetos da nossa própria classe chamada "Jogo".</p>

<h2>Principais Tópicos Abordados</h2>
<p>Durante o desenvolvimento deste projeto, cobrimos uma variedade de tópicos, incluindo:</p>

- Tratamento de Exceções em Kotlin: Utilizamos a sintaxe específica do Kotlin, como o runCatching(), para tratar exceções e lidar com erros de forma eficiente.

- Funções de Escopo e Funções de Extensão: Implementamos scope functions (funções de escopo) e extension functions (funções de extensão) para sobrescrever e estender comportamentos importantes nas classes, tornando nosso código mais modular e legível.

- Programação Orientada a Objetos: Exploramos conceitos fundamentais da programação orientada a objetos, incluindo o uso de construtores, encapsulamento, e a criação de objetos.

- Manipulação de Dados: Armazenamos os jogos em uma lista, realizando ordenação e manipulação desses dados.

- Interpolação de Strings e Variáveis: Demonstramos como utilizar interpolação de strings e variáveis para tornar o código mais legível e expressivo.

- Estruturas de Controle: Utilizamos loops e condicionais para controlar o fluxo do programa e tomar decisões.

<h2>Progresso e Aprendizado</h2>
Durante a criação deste projeto, demos um grande passo em nosso aprendizado. No primeiro curso da nossa formação, abordamos diversos tópicos e conceitos de programação em Kotlin, adquirindo conhecimento valioso e prático.
